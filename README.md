# 🌐 Website Summarizer with OpenAI

A simple and powerful Python tool that takes any website URL and generates an intelligent summary using OpenAI's GPT models.

---

## ✨ Features

- Fetch and clean text from any website
- AI-powered summarization using OpenAI (GPT-4o-mini)
- Clean and readable summaries
- Token usage tracking
- Easy to use CLI interface
- Ready for web UI extension (Streamlit)

---

## 🛠️ Technologies Used

- **Python**
- **OpenAI API** (GPT-4o-mini)
- **BeautifulSoup4** - Web scraping
- **Requests** - HTTP requests
- **Tiktoken** - Token counting
